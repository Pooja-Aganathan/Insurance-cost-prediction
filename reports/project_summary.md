# Insurance Cost Prediction – Project Summary

## Objective

Predict insurance medical expenses using machine learning regression models.

## Workflow

- Data Cleaning
- Duplicate Removal
- Outlier Analysis
- Encoding
- Scaling
- Model Training
- Hyperparameter Tuning
- Model Comparison

## Models Used

- Linear Regression
- Random Forest
- XGBoost
- LightGBM

## Key Findings

- Smoking significantly affects insurance costs
- BMI strongly influences healthcare expenses
- Tree-based ensemble models outperform baseline regression

## Business Impact

- Better insurance pricing strategy
- Risk estimation improvement
- Data-driven premium decisions
